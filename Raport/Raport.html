1.Introducere

Driver master este o aplicatie web ce ofera suport pentru invatarea semnelor rutiere si a regulilor de circulatie din Romania, de asemenea ofera si sfaturi pentru circulatia in alte tari europene. Pentru fiecare semn se pun la dispozitie informatii cu privire la semnificatie, aspect si categorie. De asemenea se poate consulta atat legislatia cat si conduita preventiva, oferite sub forma de capitole.
Sistemul va monitoriza progresul inregistrat de fiecare utilizator in procesul de invatare, in urma rezolvarii unui set de teste, structurat pe niveluri de dificultate: usor, mediu si greu.

3.Procesul de constructie

3.1.Schitare

  • Fiecare cerinta a fost mai intai schitata ca mai apoi sa se realizeze imagini model pentru produsul final, dupa care au fost impartite sarcini fiecarui membru al echipei.

3.2.Implementarea

  • PHP

    • Este utilizat pentru conectarea la baza de date, preluarea/inserarea informatiilor din/in tabele, crearea mecanismului de selectare a raspunsului corect, de selectare a numarului intrebarii curente din cadrul testului, de selectare a utilizatorului conectat in sesiunea curenta, de creare a graficelor ce retin progresul utilizatorului.

  • JavaScript

    • Este utilizat pentru a extinde sectiuni si pentru a afisa imagini la o dimensiune mai mare cat si pentru a deschide anumite sectiuni cu informatii sub forma unor mini-ferestre.

  • CSS

    • Este utilizat pentru design-ul aplicatiei:

    • ajustarea dimensiunii textului/imaginilor;
    • asezarea continutului;
    • adaugarea de imagini/fonturi pentru background si diverse elemente;
    • creearea unor campuri/mini-ferestea;

  • HTML

    • Elementele HTML sunt utilizate pentru crearea tuturor componentelor aplicatiei.

  • Baza de date

    • Este utilizata pentru stocarea informatiilor despre utilizator (id, username, e-mail, password, punctaj); intrebarile si raspunsurile testelor.
